# Selectividad · PAU (Comunitat Valenciana)

Archivo **completo** de exámenes de las Pruebas de Acceso a la Universitat (PAU / Selectividad) de la Comunitat Valenciana, **de todas las asignaturas** — las imparta o no el profesor.

Cada asignatura incluye, donde existen: **examen**, **criterios de corrección** y **modelo 2026**.

- **Convocatorias:** 2010–2026 (ordinaria + extraordinaria + extraordinaria-DANA) y modelos de 2026.
- **Fuente oficial:** [universitats.gva.es](https://universitats.gva.es/es/proves-acces-universitat).
- **Cómo navegar:** haz clic en una asignatura para entrar; usa la flecha de volver del recuadro para salir.
