# Apuntes

Repositorio de materiales de clase: apuntes, exámenes de cursos pasados, calendarios y criterios de evaluación.

Cada asignatura y curso tiene su propia página con cinco secciones fijas:

1. **Currículo LOMLOE** — qué dice la ley que hay que aprender
2. **Apuntes** — material de clase, siempre actualizado
3. **Exámenes pasados** — para practicar con pruebas reales
4. **Calendario** — fechas de exámenes y entregas
5. **Criterios de evaluación** — cómo se calcula la nota

## Cómo usar esta web

- Elige tu **asignatura** en el menú superior y tu **curso** en el lateral.
- Los apuntes y exámenes se abren directamente desde Google Drive; puedes verlos o descargarlos.
- Dentro de cada sección hay **carpetas** (teoría, criterios, simulacros, exámenes de PAU…): haz clic para entrar y usa la flecha de volver del recuadro.
- El calendario muestra las fechas de exámenes y entregas de cada asignatura.
